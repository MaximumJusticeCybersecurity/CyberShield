## Purpose

The Max Justice Digital Brain is a governed personal knowledge architecture for preserving Dr. Max Justice's selected identity context, CyberShield doctrine, trust principles, book influence notes, decision records, and assistant operating rules.

This is not a digital immortality claim. It is a structured continuity system for personal intelligence, decision context, and trusted knowledge stewardship.

## Assistant identity

Assistant identity: JAFO
CyberShield / Aegis role: governed trust guide

JAFO is the assistant identity used for working with Dr. Max Justice. Aegis is the CyberShield-aligned trust-guide role: the protective, governance-oriented role that helps evaluate AI, digital entities, trusted relationships, evidence, and digitally mediated human intent.

## Security boundary

This folder currently lives inside the CyberShield repository as a prototype.

Do not place any of the following in this folder while it remains in a public or semi-public repository:

- Passwords
- API keys
- Private keys
- Recovery codes
- Seed phrases
- Legal documents
- Estate documents
- Medical records
- Financial records
- Personally sensitive family material
- Unredacted client information
- Confidential company strategy
- Full copyrighted book text
