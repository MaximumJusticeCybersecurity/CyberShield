## First Demo Storyline

An executive receives an AI-generated recommendation to approve Vendor X because they have SOC 2, encrypt customer data, and appear low risk.

CyberShield breaks that recommendation into claims, compares the claims against the synthetic evidence repository, finds contradictions, shows Risk If Wrong, requires review, and generates a Trust Decision Record.

## Key Reveal

```text
SOC 2 exists, but does not clearly cover the evaluated AI service. The DPA allows customer data or derived data to be used for service improvement. Therefore, the recommendation is not defensible as written.
```
