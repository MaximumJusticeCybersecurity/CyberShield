# CyberShield Pre-Release Validation Checklist

## Branding

- [ ] Official MJC Logo 2026.png is used
- [ ] No placeholder logo appears
- [ ] No CS abbreviation appears as a logo
- [ ] Logo is visible, premium, and unobstructed

## Executive UX

- [ ] Executive value is understandable in under five minutes
- [ ] Landing page communicates Cyber Decision Intelligence
- [ ] Dashboard feels calm, premium, and operational
- [ ] No cyberpunk or gamer aesthetic regression
- [ ] Typography is high contrast and readable

## Action Engine

- [ ] Top actions are prioritized
- [ ] Every action has an owner
- [ ] Every action has urgency
- [ ] Every action has rationale
- [ ] Every action has expected impact

## Advisor Pane

- [ ] Metric clicks update advisor content
- [ ] Risk clicks update advisor content
- [ ] Action clicks update advisor content
- [ ] Advisor explains what this means, why it matters, next action, and MJC value
- [ ] No static placeholder advisor behavior remains

## Governance Memory

- [ ] Governance timeline displays
- [ ] Evidence age concept is present
- [ ] Ownership continuity concept is present
- [ ] Risk history concept is present

## AI Governance

- [ ] AI exposure is visible
- [ ] Shadow AI is visible
- [ ] Prompt leakage is visible
- [ ] Control boundary language is present
- [ ] Human oversight and evidence retention are present

## Reporting

- [ ] Reporting center displays required reports
- [ ] Reports can be previewed or printed
- [ ] Report layout is executive-readable
- [ ] Assumptions and limitations are planned for production hardening

## Responsive Stability

- [ ] Desktop widescreen layout has no overlaps
- [ ] Laptop layout has no overlaps
- [ ] Tablet layout stacks properly
- [ ] Mobile layout remains readable

## Deployment

- [ ] npm install succeeds
- [ ] npm run build succeeds
- [ ] dist folder deploys to GitHub Pages
- [ ] No sensitive data or secrets are committed
