# @maximumjustice/trusted-agent-spine

Private internal package candidate, version `0.1.0`.  It exposes stable builders and validators for Context Pack, Source/Citation, Missing Evidence, Human Gate, and Agent Work Receipt records.  The package structures decision evidence and stops at accountable human review.  It grants no credentials, network access, authority, or external action.

Current distribution is a locally generated, SHA-256-recorded tarball used by controlled repository consumers.  Public npm publication is prohibited.  Private GitHub Packages activation requires a separate exact-action decision and package credential design.

The single canonical implementation source is `src/core.js`.  CyberShield's prior path is a compatibility re-export only.
